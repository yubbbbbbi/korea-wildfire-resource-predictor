# ⌨️ Java Syntax Mastery: 코드 타자 분석 시스템
**멀티스레드와 실시간 검증 알고리즘을 활용한 자바 프로그래밍 특화 타자 연습 엔진**

## 프로젝트 소개
- **문제 정의**: 기존 타자 연습은 일반 문장 위주로 구성되어 있어, 개발자에게 필수적인 자바 예약어, 특수 기호(`{`, `}`, `;`), `CamelCase` 등 코딩 컨벤션 숙달에는 한계가 있음
- **솔루션**: 자바 문법 단위의 데이터셋을 구축하고, 멀티스레드 기반의 분석 엔진과 파일 저장 시스템을 통해 학습자의 타이핑 성과를 체계적으로 관리함



## 주요 구현 기술

### 1. 멀티스레드 기반 실시간 성능 분석
- **비동기 타수 계산**: 메인 로직과 분리된 별도의 스레드에서 실시간 타수(WPM/CPM) 및 경과 시간을 계산하여, 입력 흐름에 방해를 주지 않는 매끄러운 분석 환경을 구현했습니다.
- **Dynamic Calculation**: 실시간으로 입력량 추이를 추적하여 정확도와 속도 지표를 동적으로 업데이트합니다.

### 2. 타자 검증 로직
- **입력 검증**: 사용자의 각 키 입력(`KeyEvent`) 시점마다 원본 코드 스트림과 입력 버퍼를 즉각 비교 분석하여, 단일 문자 단위의 오타 감지 및 실시간 정확도 산출을 수행합니다.

### 3. 파일 기반 기록 저장
- **데이터 로깅 및 영구 저장**: 연습이 종료되면 점수, 정확도, 타수, 날짜 등의 성과 데이터를 외부 파일로 내보내어 저장합니다.
- **기록 관리**: 프로그램 실행 시 저장된 파일을 불러와 과거의 연습 기록을 유지하고 사용자의 숙련도 성장 추이를 연속적으로 확인할 수 있도록 설계했습니다.

## 작동 메커니즘
1. **데이터 로드**: 자바 문법(짧은 코드/긴 코드) 연습용 텍스트 데이터를 시스템으로 로드
2. **멀티스레드 가동**: 입력과 동시에 비동기적으로 타수 및 정확도 계산 스레드 실행
3. **실시간 검증**: 키 입력 단위의 원자적 검증 로직을 통한 오타 판별
4. **기록 저장**: 최종 연습 성과를 계산하여 로컬 파일 시스템에 데이터 기록 및 저장
