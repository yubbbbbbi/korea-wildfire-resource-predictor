# 산불 대응 자원 예측 프로그램 (Wildfire Resource Predictor)
> **실시간 기상 데이터와 과거 사례 분석을 통한 최적의 산불 대응 인력 예측 서비스**

사용자가 산불 발생 위치를 입력하면 실시간 기상 정보와 지형 데이터를 분석하여, 최적의 대응 인력 규모를 예측하고 의사결정을 지원하는 데이터 기반 서비스입니다.



## 프로젝트 소개
- **문제 정의**: 산불 발생 시 초기 대응 인력과 자원 배분에 대한 객관적인 기준이 부족하여 효율적인 진화 작업에 어려움이 있음
- **솔루션**: 다항 회귀 모델(Ridge)과 AVL 트리를 활용하여 과거 사례를 분석하고, 실시간 위치 기반 데이터를 결합해 최적의 동원 자원량을 산출함

## 주요 구현 기술

### 1. 데이터 기반 자원 예측 모델링
- **다항 회귀 모델 (Ridge Regression)**: 습도, 온도, 최대 풍속, 소방서와의 거리 등 속성을 학습하여 필요 인력 수를 예측합니다.
- **위치 기반 데이터 처리**: KakaoMap API를 연동하여 입력된 주소의 위도, 경도 변환 및 해당 지역의 고도 정보를 자동으로 추출합니다.

### 2. 효율적인 과거 사례 탐색 알고리즘 (자료구조)
- **AVL 트리 활용**: 단순 리스트 탐색($O(n)$)의 한계를 극복하기 위해 자가 균형 이진 탐색 트리인 **AVL 트리**를 설계하여 탐색 속도를 $O(\log n)$으로 최적화했습니다.
- **유사 사례 매칭 로직**: 예측된 자원량과 인접한 범위(예: 예측값 ±10) 내의 과거 실제 투입 사례들을 트리에서 조회하여 실제 투입 범위(최소~최대)를 시각적으로 제시합니다.

## 작동 메커니즘
1. **사용자 입력**: 화재 발생 위치(주소) 입력
2. **데이터 처리**: 주소 기반 좌표 추출 및 지형 데이터(고도 등) 수집
3. **예측 수식 계산**: 학습된 회귀 모델에 변수를 대입하여 예상 인력 산출
4. **AVL 트리 검색**: 산출된 예측값과 유사한 과거 실제 데이터들을 트리 순회를 통해 필터링
5. **결과 출력**: 예상 자원량 및 실제 투입 범위(최소/최대/평균) 시각화 제공
